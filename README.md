# createmyapp-iview

> iView Project Starter for `create-my-app` cli.

## Usage

``` bash
$ create-my-app init iview <project name>
```

## Features

  * equipmented with **Vue**, **Vuex** and **iView**.
  * **fetch** for ajax.
  * **sass** preprocessor.
  * **eslint** (standard)
  * **stylelint**
  * **service-mocker** for data mocking.
